Data Analysis Agent

An advanced AI-powered Data Analysis Agent that compresses dataset schemas and analysis history to reduce token usage while automating exploratory data analysis (EDA).

## Features

- Schema Compression Engine
- Token-Aware Memory Compression
- Automated Exploratory Data Analysis
- LLM-Based Analysis Planning
- Modular Agent Architecture

## Token Optimization Strategy

- Only compressed schema sent to LLM
- No raw dataset transmitted
- History limited to 5 compressed entries
- Token-aware truncation using tiktoken
